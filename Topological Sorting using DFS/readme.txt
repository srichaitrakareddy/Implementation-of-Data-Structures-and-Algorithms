How to compile and run the code:
-------------------------------
	1) The command prompt path should be in "src" directory
	2) javac -cp . sxk180037/*.java
	3) java  -cp . sxk180037/DFS
	4) For every entry please hardcode the values of vertices and edges in this manner:
		n m    v1 v2 w    v1 v2 w    0
		where,
		n: number of vertices,
		m: number of edges,
		v1: source vertex,
		v2: destination vertex,
		w: weight of edge and
		0: marks the end of input.

Note:
-----
The program will return a null if the graph is undirected.
